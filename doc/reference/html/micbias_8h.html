Control of Microphone bias hardware.

Parameters
micWhich microphone bias pin to configure
keyWhich aspect of the microphone bias to configure
valueWhich value to use
Returns
TRUE if the "mic" parameter is valid and the input value is in valid range, else FALSE.

Note: The dedicated mic bias pin is only present on some BlueCore variants. Only chips from BC5 onwards have this feature. In addition at least one ADC or DAC on chip must be enabled for the MicBias hardware to operate.

Consult the datasheet for your BlueCore variant for more information.
