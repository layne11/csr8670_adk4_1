Detailed Description

Configuration parameters for the USB traps.

In previous BlueLab versions, the bEndpointAddress field of the EndPointInfo structure had to be one of the defined addresses (see "end_point_int_out" through to "end_point_bulk_out4"), so the bEndpointAddress for an endpoint of a particular transfer type was limited to a particular set of values. This restriction has been removed, an application is now free to assign any valid address to an endpoint irrespective of the transfer type. The defined addresses can still be used and so are included for backward compatibility. New applications may prefer to specify an endpoint address as the bitwise or of the desired endpoint number with a direction mask (end_point_to_host / end_point_from_host).

The bEndpointAddress value is also used to identify the endpoint in a call to StreamUsbEndpointSource and StreamUsbEndpointSink.

Enumeration Type Documentation

enum usb_attached_status

Type reported by trap UsbAttachedStatus(void). Is the USB connection currently detached or attached, and if attached, what is it attached to. The possible values that can be returned depend on whether USB charger detection is enabled (see PSKEY_USB_CHARGER_DETECTION_ENABLE).

If usb charger detection is not enabled then can only report DETACHED or HOST_OR_HUB. If Resistive Detection is enabled then can report DETACHED, HOST_OR_HUB or DEDICATED_CHARGER. (Can not distinguish between HOST_OR_HUB and HOST_OR_HUB_CHARGER with Resistive Detection). If Voltage Source Detection is enabled then can report DETACHED, HOST_OR_HUB, DEDICATED_CHARGER or HOST_OR_HUB_CHARGER.

Consult the USB battery charging specification available at www.usb.org for more details on the difference between a dedicated charger, a standard host or hub, and a host or hub charger.

Enumerator
DETACHED 

Bluecore is detached from the bus.

HOST_OR_HUB 

Bluecore is attached to a standard host or hub (STANDARD DOWNSTREAM PORT).

DEDICATED_CHARGER 

Bluecore is attached to a dedicated charger (DEDICATED CHARGING PORT).

HOST_OR_HUB_CHARGER 

Bluecore is attached to a host or hub charger (CHARGING DOWNSTREAM PORT).

CHARGING_PORT 

temporary state, either dedicated_charger or host_or_hub_charger.
