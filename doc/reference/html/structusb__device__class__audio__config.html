Detailed Description

Structure to pass as params when calling UsbDeviceClassConfigure to set USB_DEVICE_CLASS_CONFIG_AUDIO_INTERFACE_DESCRIPTORS. Interface descriptors must be set prior to calling UsbDeviceClassEnumerate if the application wishes to use custom descriptors, otherwise defaults will be used. This structure must contain all descriptors required for the enumerated audio device classes. Terminal and unit ID's must match those used by the library, i.e. Microphone(4) -> Feature Unit(5) -> USB Output Terminal(6) USB Input Terminal(1) -> Feature Unit(2) -> Speaker(3)
