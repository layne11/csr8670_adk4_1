Detailed Description

GATT Characteristic Value Notification: Notification sub-procedure.

This message is received when a server is configured to notify a Characteristic Value to a client without expecting any Attribute Protocol layer acknowledgment that the notification was successfully received.

This is an unsolicited message which does not require response.
