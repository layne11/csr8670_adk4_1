Detailed Description

This message is sent in response to a request to the AG to dial a number from a particular memory location.

hfp_lib_status HFP_DIAL_MEMORY_CFM_T::status

If the AG responds with OK to the AT command, status is set to hfp_success. If the AG responds with ERROR or the profile instance is in the wrong state to send out the command the status will be set to hfp_fail.
