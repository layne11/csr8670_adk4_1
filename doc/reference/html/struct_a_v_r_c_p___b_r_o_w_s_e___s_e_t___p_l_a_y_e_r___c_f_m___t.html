Detailed Description

This message is generated as a result of a call to AvrcpBrowseSetPlayerRequest().

On success it returns the browsed players current folder in the folder_path field of this message. It contains the path of the folder in the format like, pairs of length of folder name and folder name repeated together "folder_depth" times, each pair representing one folder level. e.g: folder path A/BC/DEF is represented in UTF8 as 0x0001, 'A', 0x0002, "BC",0x0003, "DEF".
