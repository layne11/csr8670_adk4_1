Detailed Description

This message informs the app that the audio (SCO/eSCO) has been connected or that the attempt to open a SCO/eSCO connection has failed.

Field Documentation

hfp_link_priority HFP_AUDIO_CONNECT_CFM_T::priority

The priority of the AG.

hfp_audio_connect_status HFP_AUDIO_CONNECT_CFM_T::status

If the return status from the connection library is hci_success, then status will be set to hfp_success. Otherwise it will be set to hfp_fail.

sync_link_type HFP_AUDIO_CONNECT_CFM_T::link_type

Indicates whether a SCO or eSCO link was created.

Sink HFP_AUDIO_CONNECT_CFM_T::audio_sink

If a SCO/eSCO connection has been created the audio_sink will be the corresponding sink for the SCO/eSCO data, otherwise it will be set to zero.
