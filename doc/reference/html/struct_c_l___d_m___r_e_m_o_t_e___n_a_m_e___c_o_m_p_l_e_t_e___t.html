Detailed Description

This message indicates that a call to ConnectionReadRemoteName has completed.

A connection library client may request to read the remote name of device. This message is returned in response to such a request. If the remote name was read successfully then the status field should indicate this and the message will contain the name read along with its length. NOTES: 1. The maximum length of any name read is 31 bytes. 2. DO NOT Free memory allocated to hold the remote name after use.
