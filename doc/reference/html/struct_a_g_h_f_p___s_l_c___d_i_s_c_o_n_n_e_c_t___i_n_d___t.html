Detailed Description

This message is sent when a Service Level Connection has been disconnected.

This message informs the application that the Service Level Connection has been disconnected. This disconnect can be the result of a locally initiated action, a disconnect initiated from the HF or an abnormal disconnect e.g. link loss.
