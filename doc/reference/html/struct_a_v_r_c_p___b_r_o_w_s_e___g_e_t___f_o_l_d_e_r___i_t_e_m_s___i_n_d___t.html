Detailed Description

This message is an indication to the Target on receiving GetFolderItems request from the Controller device.

The attr_list field will be set to 0 if num_attr=0 or 255. For any other value of num_attr, size of the attributes in the attr_list will be 4*num_attr. num_attr=0 is a special value to request for all attributes and num_attr=255 is a request for no attributes. All attribute IDS in the list are defined in the interface header file under "List of Media Attribute definitions".
