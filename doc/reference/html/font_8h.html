FontId FontInit ( void  )

Initalise the font subsystem.

Returns
The number of fonts found, or zero if no fonts were found (for example, if the fonts.app file wasn't found in the filesystem).

Once the font subsystem is initialised, the filesystem containing fonts.app file can never be unmounted.
