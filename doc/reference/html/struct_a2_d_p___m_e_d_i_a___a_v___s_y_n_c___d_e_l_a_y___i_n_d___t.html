Detailed Description

An unsolicited message indicating the AV Sync delay of the remote device requires an update.

This message indicates the remote device requires an update of the AV Sync delay for the configured Media stream. This should be done by calling A2dpMediaAvSyncDelayResponse with an appropriate AV Sync delay value.
