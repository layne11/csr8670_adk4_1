Detailed Description

Indication message for an incoming OBEX GET request.

On receiving this indication message, the application must call ObexGetResponse() with the response code. If the application wants to add any Object headers, it must add it by creating new Object using ObexObJNew() and adding the headers using ObexObjAddByteHdr() or ObexObjAddVariableHdr() API.

The application may use ObexObjMapHdrValue() or ObexObjFilterHdrValue() or ObexObjExtractByteHdrValue() API to extract the headers from the Source buffer. The application must not access the source buffer after calling ObexGetResponse(). The Source buffer will be deleted for receiving the next OBEX request packet after sending the response.
