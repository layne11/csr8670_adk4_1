Detailed Description

This message is sent when a remote device wishes to establish a Service Level Connection.

This message is sent to the application to indicate that a remote device (the AG) is requesting to establish a Service Level Connection to this device. No response is required to this message. If the request has been rejected then no HFP_SLC_CONNECT_CFM will be sent to the application for this indication.
