Detailed Description

This message contains the strings that the AT command parser cannot parse.

The strings that the auto generated AT command parser supports are specified in aghfp_parse.parse along with the handler function to be called once a command has been parsed. However, if the parser cannot parse a particular AT string it will pass it to the application for further processing.
